
# ngo_management_mern

## Installation 
### Step 1
Clone the App:
`git clone  https://github.com/bablubambal/ngo_management_mern.git`
### Step 2
Open the terminal  <br/>
`cd frontend` <br/>
`npm install` <br/>
`npm start`  <br/>
`cd backend`  <br/>
`npm install` <br/>
`npm install `   <br/>

### Step 3
 Open the Backend Link:
 `http://localhost:5000`
 Open the Frontend Link:
 `http://localhost:3000`
